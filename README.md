# Gradow-button

**It's library of custom buttons with cool shadow-effect**

## Getting Started
  

**Npm module**
  - npm i gradow-btn 
    
**Manual download**
  - `<link rel="stylesheet" href="<PATH>/graddow-btn/graddow-btn.min.css"/>`
    
 **CDN** 
  - https://cdn.jsdelivr.net/npm/gradow-btn@1.0.3/gradow-btn.min.css
